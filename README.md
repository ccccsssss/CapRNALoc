# CapsBAM_ac4C

---

## Descriptions

*   `./Datasets`: This folder contains three original datasets. 
*   `cgr_reverse.py`: Python script for FCGR feature encoding.
*   `circRNA_model.py`: Python script for the CapsRNALoc of circRNA path.
*   `lncRNA_model.py`: Python script for the CapsRNALoc of lncRNA path.
*   `miRNA_model.py`: Python script for the CapsRNALoc of miRNA path.
*   `train.py`: Python script for training the model.
*   `test.py`: Python script for evaluating the model on an independent test set.

---

## Requirements

*   `python 3.10`
*   `PyTorch==2.3.1`
*   `numpy==2.0.1`
